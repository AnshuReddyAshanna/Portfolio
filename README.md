# AnshuReddy Portfolio

The website is designed to present my academic background, technical skills, projects, blog posts, and professional experience in a structured, interactive manner.

The portfolio is fully responsive, theme-aware (light/dark mode), and deployed using GitHub Pages.

Website Link:https://anshureddyashanna.github.io/AnshuReddy-Portfolio/

# Overview
This portfolio serves as a central place to showcase:
- My background in Computer Science, AI/ML, and Data Science
- End-to-end projects with technical depth
- Blogs documenting experiments and learnings
- Certifications and professional experience
- A working contact form for direct communication
The focus is on clarity, structure, and usability.

# Technology Stack
## Frontend
- HTML5 – Semantic structure
- CSS3 – Grid/Flexbox layouts, theming, responsiveness
- JavaScript (Vanilla) – Interactivity, filters, toggles

## Libraries & Tools
- Font Awesome – Icons
- Google Fonts (Poppins, Clicker Script) – Typography
- Typed.js – Text animation
- Google Apps Script – Contact form backend

## Deployment
- GitHub Pages

# Key Features

## Consistent Section Layout
- All sections (About, Blogs, Projects, Contact, Certifications) follow a unified layout system for visual consistency.

 ## Light / Dark Theme Support
 - Theme variables are handled using CSS custom properties and toggled dynamically.

## Project Filtering
Projects can be filtered dynamically by category:
- Everything
- AI/ML
- Data Science
- Data Visualization
- UI/UX
- IoT

## Expandable Project Cards
Each project card includes:
- Brief description
- Read More toggle for detailed explanation
- Tech stack displayed as pill-shaped tags
- Timeline
- Direct GitHub repository link

## Blogs Section
Grid-based blog cards with images
- Summaries
- External links to Medium articles

## Certifications Section
Includes:
- Certification name
- Issuing organization
- Timeline
- Skills covered
- Credential ID
- External verification link

## Contact Form 
- Users can send messages directly from the site
- Messages are stored via Google Apps Script
- Success feedback is shown after submission
